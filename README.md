# 喋血复仇卡组管理工具
# 导入方式：

1：python字典字符串导入
格式：{“卡组名称”:["卡牌名称"，“卡牌名称”]}

2：通过OCR识别卡组导入

支持一键导入游戏

目前仅仅支持16:9分辨率且小于等于1920*1080

导入游戏前请把分辨率改为16:9 小于1920*1080 窗口化/无边框
